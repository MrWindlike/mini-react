## About
This project is the mini version of React.

It implements these feature of React:
- [ ] The `createElement` Function
- [ ] The `render` Function
- [ ] Concurrent Mode
- [ ] Fibers
- [ ] Render and Commit Phases
- [ ] Reconciliation
- [ ] Function Components
- [ ] Hooks
