## About
This project is the mini version of React.

It implements these feature of React:
- [x] The `createElement` Function
- [x] The `render` Function
- [x] Concurrent Mode
- [ ] Fibers
- [ ] Render and Commit Phases
- [ ] Reconciliation
- [ ] Function Components
- [ ] Hooks
